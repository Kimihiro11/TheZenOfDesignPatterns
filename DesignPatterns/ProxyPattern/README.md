## 代理模式

### 定义
+ 为其他对象提供一种以控制这个对象的访问。

### 优点
+ 职责清晰
+ 高扩展性
+ 智能化

### 使用场景
+ 例如打官司需要找律师
+ Java 的Spring AOP 是个非常典型的动态代理
+ Python 的 Flask 、Tornado 


### 扩展
+ 普通代理
  + 客户端只能访问代理角色，不能访问真实角色
+ 强制代理
  + 必须通过真实角色查找到代理角色，否则不能访问
  + 高层模块new了一个真实对象结果返回了代理角色
+ 个性换代理
  + 为目标对象的方法进行拦截和过滤
+ 动态代理
  + 动态道理在实现阶段不用关心代理谁，而是在运行阶段才指定代理哪一个对象
